Políticas de Privacidade e Cookies - MVP Athena

Em conformidade com a Lei nº 22/11, de 17 de Junho, de Proteção de Dados Pessoais de Angola

1. Introdução

A PionAI, através do seu MVP Athena, respeita a privacidade dos seus usuários e está comprometida com a proteção dos dados pessoais em conformidade com a Lei nº 22/11, de 17 de Junho, de Proteção de Dados Pessoais de Angola. Esta política descreve como coletamos, usamos, protegemos e compartilhamos informações pessoais relacionadas ao uso do MVP Athena.

2. Sobre o MVP Athena

A Athena é um assistente virtual de automação residencial desenvolvido pela PionAI. Este MVP coleta e processa dados pessoais apenas no estritamente necessário para fornecer os serviços de automação residencial.

3. Dados Pessoais Coletados

Em conformidade com a Lei nº 22/11 de Angola, o MVP Athena coleta apenas os dados pessoais estritamente necessários para o funcionamento da automação residencial. São eles:

  • Identificação: Nome, e-mail, telefone (quando fornecidos no momento da configuração)
  • Uso: Comandos de voz, preferências de automação e horários de interação
  • Localização: Endereço da residência (para ativar rotinas automatizadas)
  • Dispositivos: Informações sobre os aparelhos conectados ao sistema Athena
  • Serviço: Registros internos de uso para melhoria e manutenção do sistema

Esses dados são coletados durante a instalação e uso contínuo do dispositivo, de forma local.

4. Base Legal para Tratamento de Dados

O tratamento de dados pessoais pelo MVP Athena se baseia em:

  • Consentimento explícito do usuário
  • Execução do contrato de serviços
  • Interesse legítimo da PionAI em fornecer serviços de automação

5. Uso dos Dados

Os dados pessoais coletados são utilizados exclusivamente para:

  • Oferecer e melhorar os serviços de automação residencial
  • Personalizar a experiência do usuário com o Athena
  • Manter a segurança do sistema de automação
  • Entrar em contato para atendimento e suporte
  • Atualizar o sistema com novas funcionalidades

6. Cookies

Utilizamos cookies para:

  • Melhorar a experiência de navegação
  • Manter a sessão do usuário
  • Coletar dados de análise
  • Personalizar conteúdo

7. Medidas de Segurança

Implementamos medidas de segurança para proteger seus dados, incluindo:

  • Criptografia de dados
  • Proteção contra acesso não autorizado
  • Procedimentos de backup
  • Monitoramento de segurança

8. Direitos do Titular dos Dados

Em conformidade com a Lei nº 22/11 de Angola, você tem os seguintes direitos:

  • Acesso aos seus dados pessoais
  • Retificação de informações incorretas
  • Exclusão de dados pessoais
  • Portabilidade dos dados
  • Objeto ao tratamento de dados
  • Reclamar à autoridade de proteção de dados
